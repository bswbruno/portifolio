## 📌 Sobre o Projeto

O objetivo deste portfólio é reunir meus principais trabalhos, competências e informações de contato em uma interface moderna, responsiva e intuitiva.

O site foi pensado para proporcionar uma boa experiência tanto em desktop quanto em dispositivos móveis.
